# Odin Kittens

This is a practice Rails app for learning API operations with kittens.

## Features
- Add new kittens
- View kittens
- Edit kittens
- Delete kittens

## Setup
1. Clone this repo
2. Run `bundle install`
3. Run `rails db:migrate`
4. Start the server with `rails server`
